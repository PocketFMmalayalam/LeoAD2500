##Leo AD 2500

കൊല്ലവർഷം 2500, ലിയോ എന്ന 20 വയസ്സുകാരൻ കുടുംബത്തിന്റെ കടബാധ്യതയെ തുടർന്ന് ആത്മഹത്യ ചെയ്യാൻ തീരുമാനിക്കുന്നു. അപ്രതീക്ഷിതമായി അവൻറെ ജീവിതത്തിലേക്ക് പ്രൊഫസർ ബെഞ്ചമിൻ എന്ന സയൻറിസ്റ്റ് എത്തുകയും, അവനെ മരണത്തിൽ നിന്നും രക്ഷിക്കുകയും ചെയ്യുന്നു. അയാൾ അവനെ ജീവിതത്തിലും രക്ഷിക്കാം എന്നു വാഗ്ദാനം നൽകി അയാളുടെ അപകടകരമായ ഒരു ഗവേഷണത്തിന് വിധേയനാക്കി, സൈഫർ എന്ന അന്യഗ്രഹത്തിലേക്ക് അയക്കുന്നു. മനുഷ്യരാശി ഇതുവരെ കണ്ടിട്ടില്ലാത്ത ഒരു നിഗൂഢ ലോകം തന്നെയായിരുന്നു അത്. ഭൂമിയിൽ നിന്നും കോടിക്കണക്കിന് മൈലുകൾ അകലെയാണ് പ്ലാനെറ്റ് സൈഫർ. അവിടെ പരിണമിക്കുന്ന ആക്രമകാരികളായ വിചിത്ര മൃഗങ്ങളെ വേട്ടയടിയാൽ മാത്രമേ അവിടെ ജീവിക്കാൻ കഴിയുകയുള്ളൂ. പ്രൊഫസർ ബെഞ്ചമിന്റെ വാക്കുകൾ വിശ്വസിച്ച് ലിയോ ഇതൊന്നുമാറിയതെ സൈഫറിലേക്ക് പോകുന്നു. ഈ പ്രതിസന്ധികളെ അതിജീവിച്ച് ലിയോ ഭൂമിയിലേക്ക് മടങ്ങുമോ..? അവന് വീണ്ടും അവന്റെ അമ്മയുടെയും അനുജത്തിയുടേയും ഒപ്പം കഴിയാൻ ആകുമോ..? അതോ ജീവിതത്തിന്റെ വെല്ലുവിളികളെയും അപകടങ്ങളെയും അതിജീവിച്ച് ലിയോ സൈഫർ ഗ്രഹത്തിന്റെ തന്നെ ഹീറോ ആയി മാറുമോ? കൂടുതൽ അറിയാൻ കേൾക്കൂ ലിയോ A D 2500 , പോക്കറ്റ് എഫ്എമ്മിൽ മാത്രം.
